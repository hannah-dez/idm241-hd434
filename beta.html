Rules

Save Default

  • Hover On- color shifts darker, cursor changes to finger, button size pulses- increasing by 10% then decreasing
  • Hover Off- color returns, finger turns to cursor
  • Mouse Down
    • btn opacity decreases by 30%
    • btn size decreases by 10%
  • Mouse Up
    • Btn size increases to origional size
    • White btn icon fades out
    • White btn Icon slams down
    • Btn particles burst from button, white then fade to red
    • Opacity of effect and button decrease by 100%
    • Btn fades in with transparent background color, the icon and the border are red
    • Shortcut panel pops up at the bottom; exits screen after 3 seconds
  • Database save flag set TRUE

Save Active

  • Hover On- Curser Changes to finger, button color changes to grey and icon is white
  • Hover off- Curser changes back, red border and icon return
  • Mouse down- btn size decreases by 10%, grey turns to black
  • Mouse Up
    • Icon changes back to white, button turns red
    • btn size increases to origional size
    • Removed panel pops up at the bottom; exits screen after 3 seconds
  • Database Save Flag set to FALSE
