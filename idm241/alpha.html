Rules

Save Default

  • Hover On- color shifts darker
  • Hover Off- color returns
  • Mouse Down
    • btn opacity decreases by 30%
    • btn size decreases by 30%
  • Mouse Up
    • Btn text changes to Saved
    • Btn color changes to black
    • Btn Size increases to origional size
  • Database save flag set TRUE

Save Active

  • Hover On- Curser Changes to finger
  • Hover off- Curser changes back
  • Mouse down- btn size decreases by 30%
  • Mouse Up
    • Text changes back to save
    • btn size increases to origional size
  • Database Save Flag set to FALSE
