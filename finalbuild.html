Rules

Pin Div

  • Hover On
    • Cursor changes to finger
    • Grey overlay appears over image
    • Board selector fades from 0% to 100% opacity
    • Pin btn scales up from 0 to 100%
    • Share and More buttons wave in from the bottom before settling
  • Hover Off
    • Finger changes back to cursor
    • Pin btn scales down to 0%
    • Share and More Buttons fall out of frame
    • Overlay and board selector fade out

Pin Button Default

  • Hover On- color shifts darker, cursor changes to finger, button size pulses- increasing by 10% then decreasing
  • Hover Off- color returns, finger turns to cursor
  • Mouse Down
    • btn opacity decreases by 30%
    • btn size decreases by 10%
  • Mouse Up
    • Btn size increases to origional size
    • White btn icon fades out
    • Btn Flashes Grey, then red circle fills it before swiping away
    • Btn fades in with transparent background color with a red border
    • Shortcut panel scales and fades up at the bottom; scales down and fades out after 3 seconds
  • Database save flag set TRUE

Pin Button Active

  • Hover On- Curser Changes to finger, button color changes to grey and border disappears
  • Hover off- Curser changes back, red border returns
  • Mouse down- btn size decreases by 10%, grey turns to black
  • Mouse Up
    • Button turns red
    • btn size increases to origional size
    • Removed scales and fades up at the bottom; scales down and fades out after 3 seconds
  • Database Save Flag set to FALSE

Shortcut Panel

  • Hover On
    • Div size increases by 10%
    • Cursor changes to finger
    • Hide div animation is paused
  • Hover Off
    • Div size decreases by 10%
    • Finger changes back to cursor
    • Hide div animation is resumed

Removed Panel

  • Hover On
    • Div size increases by 10%
    • Cursor changes to finger
    • Hide div animation is paused
  • Hover Off
    • Div size decreases by 10%
    • Finger changes back to cursor
    • Hide div animation is resumed
